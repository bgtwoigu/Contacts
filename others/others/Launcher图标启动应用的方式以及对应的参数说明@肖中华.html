         2.部分部分模块命名不规范，请查看印象笔记中Common分类下的“git库代码”一文中的“包的命名原则 ”项。

音频播放
集成到launcher中，直接启动。
Intent intent = new Intent(context , arg0);    
intent .setAction(arg1);        
context .startActivity(intent);
arg0:启动的目标Activity的详细路径；

arg1:传递的action字符串。
事项提醒
作为APK内置到ROM中，需要提供的信息包括：包名和主Activity的详细路径
Intent intent = new Intent(Intent. ACTION_MAIN);
intent.addCategory(Intent.CATEGORY_LAUNCHER);
ComponentName comp = new ComponentName(arg0 ,                             arg1 );
intent.setComponent(comp );
context.startActivity(intent );
arg0为事项提醒的包名：com.yongyida.robot.notification

arg1为事项提醒主Activity的详细路径：com.yongyida.robot.notification.MainActivity。
电话
使用系统应用
Intent intent = new Intent();
intent.setAction(Intent.ACTION_DIAL);
context.startActivity(intent );

算术
作为APK内置到ROM中，需要提供的信息包括：包名和主Activity的详细路径
Intent intent = new Intent();
intent.setAction(arg0);
intent.setFlags(Intent.FLAG_INCLUDE_STOPPED_PACKAGES);
context.sendBroadcast(intent );
arg0为启动算术的Action附带值：com.yydrobot.QUSETION
设置
使用系统应用
Intent intent = new Intent();
ComponentName comp = new ComponentName(arg0 , arg1 );
intent.setComponent(comp );
intent.setAction("android.intent.action.VIEW" );
context.startActivity(intent );
arg0为设置的包名：com.android.settings

arg1为设置主Activity的详细路径：com.android.settings.Settings 。
