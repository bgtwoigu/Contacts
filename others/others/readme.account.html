readme.account

Http

匿名登录
/account/login/anno

req
unique_id            string        唯一id

rsp
ret                         int                  0 OK, 非0 失败
id                         long          id和session为登录状态，需要保存起来,如果本地有此数据则不需要登录,默认缓存1个月
session                    string        使用此数据能才能做后续操作

----------------------------------------

通过旧密码修改账号密码
/account/update/password/oldPwd

req
unique_id            string        唯一id
oldPwd               string        旧密码
newPwd               string        新密码

rsp
ret                         int                  0 OK, -1 传入参数为空，-2 唯一id(6-100)或新密码(6-32)长度不符合，-3 唯一id错误，1 旧密码错误，2 服务器更新账号密码失败
msg                  String      服务器反馈信息
session                    string        使用此数据能才能做后续操作


------------------------------------------
通过验证码修改账号密码
/account/update/password/verify

req
unique_id            string        唯一id
phone                string        手机号码
verify               string        验证码
newPwd               string        新密码
rsp
ret                         int                  0 OK,-1 传入参数为空，-2 唯一id(6-100)或新密码(6-32)长度不符合，-3 唯一id错误，-4 手机号码不是绑定的手机号码   1 验证码过期， 2 验证码错误，3 服务器更新账号密码失败
msg                  String      服务器反馈信息
session                    string        使用此数据能才能做后续操作

session                    string        使用此数据能才能做后续操作
