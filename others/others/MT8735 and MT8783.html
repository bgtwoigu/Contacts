MT8735 and MT8783

MT8735 64-bit quad-core LTE tablet platform

MediaTek MT8735 is a highly integrated LTE System-on-Chip (SoC) which incorporates advanced features, e.g. LTE cat.4, Quad core operating at 1.3GHz, 3D graphics (OpenGL|ES 3.0), 13M camera ISP, LPDDR3-640 Mbps, WXGA display and 1080p video codec. MT8735 helps tablet manufacturers build high-performance LTE smart device with PC-like browser, 3D gaming and cinema class home entertainment experiences.

 

Features

  • ARM Cortex-A53 64-bit quad-core processor
  • ARM Mali-T720 MP2 GPU
  • 4G FDD LTE, TDD LTE, HSPA+, TD-SCDMA, CDMA200 and EDGE
  • 1280 x 800 display
  • 16MP image signal-processor
  • H.264 1080p Full HD video playback
  • Dual-band Wi-Fi, Bluetooth, GPS and FM receiver



MT8783 64-bit octa-core LTE tablet platform

MediaTek’s MT8783 is a highly integrated LTE System-on-Chip (SoC) which incorporates advanced features, e.g. LTE cat.4, Octa-core operating at 1.3GHz, 3D graphics (OpenGL|ES 3.0), 16M camera ISP, LPDDR3-1466 Mbps, WUXGA display and 1080p video codec. MT8783 helps device manufacturers build high-performance LTE smart devices with PC-like browser, 3D gaming and cinema class home entertainment experiences.

 

Features

  • ARM Cortex-A53 64-bit octa-core processor
  • ARM Mali-T720 MP3 GPU
  • 4G FDD LTE, TDD LTE, HSPA+, TD-SCDMA, CDMA200 and EDGE
  • 1920 x 1200 display support
  • 16MP image signal-processor
  • H.264 1080p Full HD video playback
  • Dual-band Wi-Fi, Bluetooth, GPS and FM receiver
