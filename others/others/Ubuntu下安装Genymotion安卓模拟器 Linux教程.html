Ubuntu下安装Genymotion安卓模拟器 Linux教程

2015年02月15日 计算机 暂无评论 阅读 13,089 次

怎么样在自己的电脑系统中安装安卓模拟器呢？这个问题应该有很多人为问。我之前也一直在找使用安卓模拟器的软件，后来在windows系统中用了一个名叫bluestacks的软件，不过不是特别好用，有些软件也是不用够运行的。

后来，我又找了很久，终于我找到了一个名叫Genymotion的软件，这个Android模拟器软件有windows和linux等版本的软件，安装也很简单，使用的时候需要到官网上注册一个账户才能够正常使用。

Genymotion官网：www.genymotion.com

安装教程：

官网下载软件，例如，你下载下来的安装文件是：genymotion-2.2.2_x64.bin,并且放在你的主目录（/home/root）下。

接下来，通过组合键ctrl + alt + T键打开终端，输入以下命令：

sudo apt-get install virtualbox

上面这一句是安装虚拟机软件Virtualbox，没有安装这个软件不能够使用Genymotion软件，Windows版本也是一样。

cd /home/root

chmod +x genymotion-2.2.2_x64.bin

./genymotion-2.2.2_x64.bin -d /home/tenfei

然后会问你是否同意安装，属于y字母，回车

安装完后，进入/home/tenfei/genymotion目录，双击运行genymotion文件即可。

接下来就是登录用户和添加你所要模拟的安卓系统，然后就可以尽情体验安卓系统的乐趣了。
