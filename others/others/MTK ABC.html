sudo add-apt-repository: command not found

I don't know about 64 bit, but on 32 bit, add-apt-repository is provded by the package python-software-properties.
You could always add the repository to /etc/apt/sources.list manually.
sudo apt-get install python-software-properties

Start/Stop MTK Logger
adb shell am broadcast -a com.mediatek.mtklogger.ADB_CMD -e cmd_name start --ei cmd_target 7
adb shell am broadcast -a com.mediatek.mtklogger.ADB_CMD -e cmd_name stop --ei cmd_target 7
(start表示开启，stop表示关闭；7可改为1/2/4，分别代表MobileLog/ModemLog/NetworkLog)
